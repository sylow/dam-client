# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-10

### Added
- Initial release
- Support for Webdam provider
- Support for Bynder provider
- Unified search interface
- Asset metadata retrieval
- Thumbnail URL generation
- Product photo filtering
- Item number-based search
- Priority-based image selection for Bynder (Product_Main)
- Automatic image type recognition
- XMP metadata extraction

### Features
- Factory pattern for provider selection
- Environment-based configuration
- Common model interface across providers
- Rails cache integration support
